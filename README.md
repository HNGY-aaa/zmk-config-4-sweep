# ZMK config for sweep

在fork库基础上魔改了一下键位，基本上是因为ZMK studio不支持对条件层的修改才大动干戈自己改了一个配置问题。

希望能正常运行。

原作者的README在_README文件中，请自行跳转。

## 修改遵循的思路

一个简单的键位设置的指导思想

- 除了主键区键位这种不好动的按键以外，把使用频率高的案件放在中间而不是边缘

- 修饰建对称放置

- 尽可能保持修改前的相对位置

- 削减使用的键数

- 在击键速度过快时的误触发问题怎么解决可以提上日程了

    + 我暂时还没有什么头绪。而且我本身打字习惯也不算好，倒是可以先适应下再说

    + 单独给游戏划了一个键区，目前还是处于摸索状态，简单的保留了左手键区的主要内容，修饰键确实没地方放了
